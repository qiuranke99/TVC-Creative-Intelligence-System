# TCIS v3 专业广告与 TVC 来源台账

状态：active evidence  
核验日期：2026-07-11  
用途：支持 TCIS v3 的术语、流程、角色、人机决策、创意方法、制作时序、claim 与 AI 边界

## 1. 证据规则

- 优先专业协会、监管机构、代理公司/集团官方材料、出版社/作者、工会和商业制作组织；
- 代理公司口号、定位、文化原则、正式方法和评价量表分别记录；
- Award entry/case 是效果叙事或案例证据，不自动证明普遍 workflow；
- 出版商简介与目录只能支持公开可见的方法主线，不补写受版权保护的书内细节；
- 地区、合同和公司差异保留，不声称存在全球唯一 SOP；
- `TCIS inference` 是基于证据的系统设计裁决，不冒充来源原话。

**术语裁决：** 没有发现 IPA、APG、Effie、Cannes、4As 或 WARC 的统一 `creative platform` 行业词典。跨来源较稳定共同点是“高于单一执行并可持续生成多个执行”。因此 TCIS 只能条件使用，不能设为 universal stage。

## 6. Commercial production、导演与后期时序

| ID | 来源 | 类型 | 能支持 | 不能支持 |
|---|---|---|---|---|
| PR-01 | [IPA/APA/ISBA Production Pitch Process](https://ipa.co.uk/knowledge/documents/production-pitch-process-initiative) | 联合行业最佳实践 | production pitch 前 script/creative 应完整并 approved for production；director-creative meeting、treatment、budget、award | 不是法律；地区合同可不同 |

| PR-04 | [AICP Standard Commercial Production Agreement](https://aicp.com/assets/editor/Standard_Commercial_Production_Agreement_1.pdf) | 合同范本 | Agency/Client 提供 scripts、storyboards、creative guidance；Producer 提供完整 production | 不等于所有项目法律合同 |
| PR-05 | [Clearcast Clearance Process](https://clearcast.co.uk/clearance-process/) | 英国广播清稿机构 | Script preproduction -> Rough Cut -> Final TVC；script 包含 visual/VO/text 与 substantiation | 仅直接适用于其市场/服务 |

**时序裁决：** 标准 agency-led TVC 中，agency concept/script/board 批准早于 director treatment；director、DP、production designer、editor 是重要 production craft owners，但不是默认上游 strategy/idea owners。

## 8. 关键反证

1. 单支六秒 demo 可以从 proposition/proof/concept/script 进入 production，不需要长期 platform。
2. 已有 platform 的新 TVC 不应每次重建 platform。
3. Film form 本身可能就是核心 idea，不能在抽象 platform 后才讨论。
4. Director 可以在 treatment 中显著改善创意，但这种贡献不是默认免费上游 ownership。
5. Animation/editor、VFX-first、AI-native 会把 craft 决策前移，但不会使 DP/PD/editor 成为品牌策略 owner。
6. 更多人类审批不自动更专业；正确做法是必要 decision owner、共同 brief、客观反馈和明确 signoff。
7. 更多候选不自动更原创；数量可以越过第一反应，但不能代替孵化、CD judgment 和 client decision。

## 9. Evidence-backed architecture decisions

| Decision | 证据基础 | 裁决 |
|---|---|---|
| universal creative platform | TM-01 至 TM-06 | 删除；仅条件分支 |
| D0 user tool authorization | 产品要求 + Codex 本地能力；不属于广告方法 | 删除用户门；本地自主，外部后果另行批准 |
| maximum three human decisions | BP-01 至 BP-08 | 删除；每个 decision-bearing artifact 交互后锁 |
| QD/search as creative core | AG/BK sources 与实践反证 | 降为后台候选治理 |
| early joint director/DP/PD/editor | PR-01 至 PR-06 | 删除默认；按 production timing 或显式例外进入 |
| Agency AD = Production Designer | 角色与制作证据 | 拆成两种能力 |
| advertising director + film director | 导演权责证据 | 合并为 Commercial Director |
| prompt approval = media approval | CR-03 + media QA 原则 | 禁止；必须选择实际媒体 |

## 10. 未知与边界

- 顶级代理公司的现行内部手册、CD review 频率和完整 proprietary workflow 通常不公开；
- Ogilvy Big ideaL、Leo HumanKind 等在 2026 年各办公室的强制程度未知；
- 合同、工会、市场和 direct-to-client/in-house 结构会改变角色名称和时点；
- Effie 与 Cannes 适合检查 challenge -> strategy -> idea -> execution -> results 逻辑，不是 owner/approval SOP；
- 这些限制不会削弱 TCIS 的根裁决：术语条件化、权责分离、先 agency creative 后 production interpretation，以及每个决策产物的人类 lock。
